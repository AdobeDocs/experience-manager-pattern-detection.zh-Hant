# INS {#ins}

無效命名空間

## 背景 {#background}

>[!CONTEXTUALHELP]
>id="aemcloud_bpa_ins_overview"
>title="無效命名空間"
>abstract="INS 會識別 AEM 執行個體的命名空間問題"

`INS` 無效命名空間會識別 AEM 執行個體的命名空間問題。

子類型用於識別不同類型的資訊，例如：

* `uri`：識別 AEM 中無效的命名空間 URI。

## 可能影響和風險 {#implications-and-risks}

* 無法複製內容 (跨層) 或複製內容 (跨環境 - 經過 `/crx/packMgr` 或內容複製)。
